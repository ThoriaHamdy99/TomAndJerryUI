# 🐭 Tom And Jerry - Jetpack Compose UI Screens

This project contains **3 UI screens** built using **Jetpack Compose** to replicate the provided **Figma design** with high visual accuracy.

## 🎯 Project Goal

> The goal of this project is **purely UI-focused**:  
> ✅ Implement the exact layout and styling from Figma using Jetpack Compose.

No backend, business logic, or navigation flow is required — only the **static UI** of three screens.
No Clean, modular Composables for each screen will be later in another project ISA.

---

## 📱 Features

- Built entirely with **Jetpack Compose**
- Mirrors **Figma design pixel-perfectly**

---

## 🛠️ Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Android Studio (Giraffe or later)**
